# Rock, Paper, Scissors, Lizard, Spock Game

O jogo Rock, Paper, Scissors, Lizard, Spock é uma versão expandida de "Pedra, Papel e Tesoura", criada para reduzir empates e aumentar a variedade estratégica. O objetivo central é derrotar o oponente (computador) selecionando uma jogada conforme um conjunto fixo de regras.
