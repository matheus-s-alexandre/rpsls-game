# UC02: Configure Match Mode (Configura Modo de Jogo)

## Objetivo
Permitir que o jogador escolha o modo de jogo entre “Partida Única” ou “Melhor de 3” (Melhor de 3 vitórias)

## Requisitos
Não há

## Atores
Jogador

## Prioridade
Não se aplica.

## Pré-condições
O jogador deve ter acesso à interface principal do jogo.

## Frequência de uso
Não se aplica.

## Criticalidade
Alta

## Condição de entrada
O jogador clicar no botão “Iniciar Jogo”

## Fluxo Principal
1. A aplicação exibe as opções de modo de jogo;
2. O jogador escolhe o modo desejado;
3. O sistema registra a escolha;

## Fluxo Alternativo
Não há

## Pós condições
Se o modo escolhido for "Melhor de 3", o mesmo deve ser considerado nas próximas rodadas.

## Regras de negócio> 
O modo “Melhor de 3” requer três vitórias para encerrar a partida. O modo “Partida Única” encerra o jogo após uma rodada.
